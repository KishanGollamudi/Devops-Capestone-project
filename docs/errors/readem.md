# Errors & Resolutions

## Overview

This directory documents **errors encountered during the project** and how they were resolved.

Each file follows this format:
- Problem description
- Error logs
- Root cause
- Resolution steps

This mirrors real-world DevOps incident documentation.

---

## Documented Issues

- SSH permission denied between Jenkins and Ansible
- Docker port already in use
- Nexus artifact not visible after upload
- SonarQube Node.js dependency error
- Jenkins credentials mismatch
- Docker daemon permission issues

---

## Learning Outcome

These issues helped reinforce:
- Linux permissions and SSH authentication
- Docker networking and port binding
- CI/CD pipeline debugging
- Credential management best practices
